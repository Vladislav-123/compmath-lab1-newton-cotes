# Вычисление определённых интегралов методами Ньютона–Котеса

Расчетно-графическая работа №1 по курсу "Математический анализ". Реализованы и исследованы составные квадратурные формулы прямоугольников (средней точки), трапеций, Симпсона и "трех восьмых".

## Описание

Проект позволяет:
- приближенно вычислять определенные интегралы с контролируемой точностью;
- исследовать зависимость числа узлов разбиения от требуемой точности;
- анализировать влияние гладкости подынтегральной функции на порядок сходимости;
- сравнивать фактические погрешности с теоретическими оценками.
